Privacy Policy

Effective Date: May 26, 2026

Overview

Safepal Menola respects user privacy and is designed to operate with minimal permissions while providing lightweight browser-based creative utility tools. The extension does not collect, sell, store, or share personal information.

Information Collection

  • No personally identifiable information is collected.
  • No passwords, payment information, or account credentials are accessed.
  • No browsing history or browsing activity is tracked.
  • No website content, form data, or keystrokes are monitored.
  • No analytics trackers, advertising identifiers, or fingerprinting technologies are used.

Permissions & Usage

  • storage — Used only to save local extension settings inside the user's browser.
  • clipboardWrite — Required only for copy functionality initiated directly by the user.
  • https://firestore.googleapis.com/* — Used only to retrieve a secure read-only Firebase configuration document.

Remote Configuration

Safepal Menola may use a secure Firebase-hosted read-only configuration file to manage extension behavior such as displaying built-in tools or approved webpages. No remote code execution is used, and no personal information is collected or transmitted.

Third-Party Services

Trusted third-party services may occasionally be used only when required for optional utility features. These services are accessed only when initiated by the user.

Data Security

All creative tools operate locally inside the browser environment. Safepal Menola does not maintain external servers for storing user-generated content, browsing activity, or personal data.

Policy Updates

This Privacy Policy may be updated periodically to reflect extension improvements, browser compatibility updates, or Chrome Web Store policy requirements.
